  1. Home
  2. Savoy Group Sharm El Sheikh Website

Project information

  • Category: Front End Development, UI UX
  • Client: Savoy Group
  • Project date: May, 2015
  • Project URL: www.savoy-sharm.com“not the current”

Tight deadline mission

Moving from and old school web design with tables and flash animation. To new responsive website with modern design “By the standards of that period”.

My job was to prepare the UX flow and user journey then makes the UI design for the website, and building the website, It’s a static with small module for online booking a JS that link to the dynamic reservation system and adding some attributes for the booking links such as hotel arrival departure dates and number of guests.

In addition to build the website SEO plan and distribute the keywords among all pages links images name and alt attributes, setting up the tracking codes for Google Analytics.

It was a challenge for me to finalize this group website with four hotels in just 2 weeks. And the mission is accomplished.
